## Selection Sort

Selection sort algorithm sorts an array by repeatedly finding the minimum element from unsorted part and putting it at the beginning.

```txt
Input  : 5 4 1 6 2
Output : 1 2 4 5 6
```

```txt
Time Complexity : O(n^2)
```

```txt
Space Complexity : O(1)
```
